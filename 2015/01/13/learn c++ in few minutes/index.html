const与指针

  • 野指针指未初始化的指针
  • 空指针可以确保不指向任何对象或者函数
  • int const *p = NULL <=> const int *p = NULL
    定义了一个指向const变量的指针，不能通过指针去修改他所指向的变量的值，但是可以改变指针的指向的变量。
  • int * const p = &a
    定义了一个const指针，注意的是，在定义的同时必须要初始化指向一个a,我们不能改变指针的指向，但是我们可以通过指针改变变量的内容。

void指针

  • 又叫无类型指针,可以指向任何类型的指针，但是在使用时必须将其进行强制转化。
    eg:
    cout << * (int *)p << endl;
q = (int *) p

new&delete

  • new用来动态申请内存。
    指针变量 ＝ new 数据类型
    指针变量 ＝ new 数据类型（初值）
    申请连续时：
    指针变量 ＝ new 数据类型[元素个数]
    正确返回一个指针类型，申请失败则返回一个空指针。

  • delete用来释放内存空间。
    delete 指针变量
    连续时用下面的，否则可能造成内存泄漏
    delete [] 指针变量

引用

  • 类型标示符 & 引用名 ＝ 目标变量名
    操作与对改变量进行操作一样，需要注意的是:
    1.不能建立void的类型的引用
    2.不能建立数组的引用
    3.可以将变量的引用的地址赋给一个指针，指向的是原来的地址
    4可以建立指针变量的引用

函数默认参数

  • 实参与形参的结合是结合方式是从左向右的，所以，带有默认值的参数必须放在形参表的右端。

函数重载

  • 函数名字相同，参数不同

内置函数

  • 在函数首行加上inline即可
