matlab 2-Dimensional Graphics

Try to show the function graphs of sinx and cosx.
Requirement: Two curves should be plotted in the same figure window, where the sine curve is shown by a solid line, the cosine by a dashed line, and they also should be shown in different colors and indicated by legend. Use grid. Set the range of x to be [-2pi, 2pi].

Plot the voltages of the three-phase alternating current with the amplitude 220V, the frequency 60Hz, and the phase difference
120 degree. (Show the waveform of two periods, along with a zero line)

Plot the following function curves:
y1=150e^(0.05x)sinx,
y2=0.8e^(0.5x)sin(10x).
Requirement:: Plot in a single figure
window. The range of x is [0,20]. Set the
ordinate of y1 to be 150 times of y2 .

Plot the curves of y=lnx in x-logarithmic scale, y-logarithmic scale and
xy-logarithmic scale, respectively. Plot them separately in 3 subplots of a same window,with different titles. Use grid. X range is [0,1000]
