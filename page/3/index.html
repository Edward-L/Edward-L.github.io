const与指针

  • 野指针指未初始化的指针
  • 空指针可以确保不指向任何对象或者函数
  • int const *p = NULL <=> const int *p = NULL
    定义了一个指向const变量的指针，不能通过指针去修改他所指向的变量的值，但是可以改变指针的指向的变量。
  • int * const p = &a
    定义了一个const指针，注意的是，在定义的同时必须要初始化指向一个a,我们不能改变指针的指向，但是我们可以通过指针改变变量的内容。
