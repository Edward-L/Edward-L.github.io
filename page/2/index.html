pip安装库时遇到的问题

Posted on 2015-04-30   |   In python   |     |    Views

昨天晚上想要新装一个库，但是发现直接sudo pip install安装提示成功，但是直接在import的时候提示没有这个库，奇怪，又试着重新安装提示已经安装过了，然后看了下他的路径发现了一些问题。

发现直接在终端里用python的时候它居然跑到了MAMP里的python，然后我直接用pip或者easy_install安装的库在/Library/Python/2.7/site-packages/所以自然就不能就调用了,看了下,MAMP中的python是一个软链接，直接改掉了MAMP中python的名字，这样他就直接调用了Library中的了，问题解决。看来还是MAMP的问题。
